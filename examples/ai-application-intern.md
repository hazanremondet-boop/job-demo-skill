# Example: AI 应用开发实习生

## 原始岗位

```text
岗位：AI 应用开发实习生

岗位职责：
1. 参与公司 AI 应用产品的设计与开发。
2. 接入大模型 API，完成文本分析、知识库问答、内容生成等能力。
3. 与产品、后端同学协作，完成前端页面和接口联调。
4. 关注 AI 应用效果，持续优化 prompt 和用户体验。

任职要求：
1. 熟悉 React/Vue 至少一种前端框架。
2. 了解 Node.js 或 Python 后端开发。
3. 了解 LLM、RAG、Prompt Engineering。
4. 有完整项目经验或可展示 demo 优先。
```

## 能力信号

- AI 应用落地：不是只会聊天，而是能接入业务流程。
- RAG/知识库：能处理私有知识、检索来源、生成回答。
- 前后端协作：能做页面、接口、状态和错误处理。
- 产品意识：能围绕用户场景设计功能。
- 可展示交付：能部署或录屏演示。

## 推荐 demo

项目名：AI 求职项目路线决策与生成工作台

定位：

> 面向低年级学生的 AI 求职工作台，输入目标岗位 JD 和个人背景后，先输出三条项目路线，帮助用户确认项目内容、产品逻辑和技术栈，再在确认后生成可运行 demo、简历 bullet 和 HR 话术。

## MVP 功能

1. JD 输入：粘贴岗位职责和要求。
2. 能力拆解：提取技术栈、业务场景、隐含能力。
3. 三路线决策：给出最快落地、岗位命中、技术展示三条路线。
4. 路线确认：对比技术栈、MVP 范围、风险和求职表达。
5. 项目生成：用户确认后生成项目结构、页面/API、mock 数据、README、简历 bullet 和 HR 开场消息。

## 技术方案

- Frontend: React / Next.js。
- Backend: Next.js API Routes 或 FastAPI。
- AI: OpenAI 兼容 API。
- Database: SQLite 或 Supabase。
- Deploy: Vercel。

## AI Prompt 方向

```text
你是一个 AI/全栈实习求职顾问。
请根据用户提供的岗位 JD，提取岗位所需能力，判断适合做什么 demo，
先输出三条 48 小时内可完成的项目路线，等用户确认后再生成完整项目。

输出必须包含：
1. 岗位能力信号
2. 三条项目路线
3. 每条路线的项目一句话定位
4. MVP 功能
5. 技术栈和选择原因
6. 路线推荐和确认问题
7. 确认后的项目生成要求
8. 简历 bullet 和 HR 开场话术
```

## 简历 bullet

面向 AI 应用实习求职场景独立开发 Job Demo Skill 工作流工具，支持岗位 JD 解析、能力信号提取、三条项目路线对比、路线确认后的项目生成与 HR 话术生成；使用 React/Next.js 与 LLM API 完成前后端和 AI 能力闭环，用于展示 AI 应用落地、产品拆解和全栈开发能力。

## HR 话术

您好，我是大二学生，正在找 AI 应用开发实习。我看到岗位里提到大模型接入、RAG/Prompt 和完整项目经验，所以做了一个 Job Demo Skill 工作流 demo：输入岗位 JD 后，它会先生成三条项目路线，确认方向后再生成可运行项目、README、简历 bullet 和 HR 话术。想用这个项目更具体地展示我对 AI 应用落地和全栈开发的理解，方便的话希望争取一次沟通机会。
