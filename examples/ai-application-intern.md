# Example: AI 应用开发实习生

## 原始岗位

```text
岗位：AI 应用开发实习生

岗位职责：
1. 参与公司 AI 应用产品的设计与开发。
2. 接入大模型 API，完成文本分析、知识库问答、内容生成等能力。
3. 与产品、后端同学协作，完成前端页面和接口联调。
4. 关注 AI 应用效果，持续优化 prompt 和用户体验。

任职要求：
1. 熟悉 React/Vue 至少一种前端框架。
2. 了解 Node.js 或 Python 后端开发。
3. 了解 LLM、RAG、Prompt Engineering。
4. 有完整项目经验或可展示 demo 优先。
```

## 能力信号

- AI 应用落地：不是只会聊天，而是能接入业务流程。
- RAG/知识库：能处理私有知识、检索来源、生成回答。
- 前后端协作：能做页面、接口、状态和错误处理。
- 产品意识：能围绕用户场景设计功能。
- 可展示交付：能部署或录屏演示。

## 推荐 demo

项目名：AI 求职 JD 分析与项目方案生成器

定位：

> 面向低年级学生的 AI 求职工作台，输入目标岗位 JD，输出岗位能力拆解、demo 方向、MVP 功能、开发计划、简历 bullet 和 HR 话术。

## MVP 功能

1. JD 输入：粘贴岗位职责和要求。
2. 能力拆解：提取技术栈、业务场景、隐含能力。
3. Demo 推荐：给出 1 个主推 demo 和 2 个备选方向。
4. 开发计划：生成 24/48/72 小时任务安排。
5. 展示材料：生成简历 bullet 和 HR 开场消息。

## 技术方案

- Frontend: React / Next.js。
- Backend: Next.js API Routes 或 FastAPI。
- AI: OpenAI 兼容 API。
- Database: SQLite 或 Supabase。
- Deploy: Vercel。

## AI Prompt 方向

```text
你是一个 AI/全栈实习求职顾问。
请根据用户提供的岗位 JD，提取岗位所需能力，判断适合做什么 demo，
并输出一个 48 小时内可完成的 MVP 项目方案。

输出必须包含：
1. 岗位能力信号
2. 推荐 demo 名称
3. 项目一句话定位
4. MVP 功能
5. 技术栈
6. 开发计划
7. 简历 bullet
8. HR 开场话术
```

## 简历 bullet

面向 AI 应用实习求职场景独立开发 JD-to-Demo 工作流工具，支持岗位 JD 解析、能力信号提取、demo 方案推荐、开发计划生成与 HR 话术生成；使用 React/Next.js 与 LLM API 完成前后端和 AI 能力闭环，用于展示 AI 应用落地、产品拆解和全栈开发能力。

## HR 话术

您好，我是大二学生，正在找 AI 应用开发实习。我看到岗位里提到大模型接入、RAG/Prompt 和完整项目经验，所以做了一个 JD-to-Demo 工作流 demo：输入岗位 JD 后，它会生成能力拆解、项目方案、开发计划和 HR 话术。想用这个项目更具体地展示我对 AI 应用落地和全栈开发的理解，方便的话希望争取一次沟通机会。
